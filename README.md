## Install

```bash
git clone git@github.com:rofixro/create-vue3-app.git

cd create-vue3-app

npm install

npm run dev
```

If you prefer to do it manually with the cleaner git history.

```bash
npx degit rofixro/create-vue3-app my-vue3-app
```
